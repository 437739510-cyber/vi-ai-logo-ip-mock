# Brand Brain — Pricing Draft V1

> **价格草案**
> 日期：2026-05-31
> 状态：DRAFT — 等待商业试点验证后调整
> 说明：草案，非上线收费系统。用于客户沟通和商业计划参考。

---

## 定价原则

```
1. 比传统品牌设计公司低 10-20 倍
2. 比通用 AI 工具高（因为包含品牌分析价值）
3. 三种模式对应不同价值层级
4. 试点期间可灵活调整
```

---

## 三种模式定价

### Professional — 专业模式

| 项目 | 内容 |
|------|------|
| 定位 | 品牌经理、设计师直接使用 |
| 输入 | 专业品牌术语 |
| 输出 | 完整品牌手册（15模块/23页） |
| 包含 | 品牌分析 + IP策略 + 视觉方向 + 生成 |
| **建议价格** | **3,000 - 5,000元/次** |
| 试点价 | 1,000 - 2,000元 |

### Assisted — 辅助模式

| 项目 | 内容 |
|------|------|
| 定位 | 中小企业主，有基本品牌意识 |
| 输入 | 补全+确认流程 |
| 输出 | 核心品牌手册（8-12模块） |
| 包含 | 品牌分析 + 模块规划 + 生成 |
| **建议价格** | **1,000 - 3,000元/次** |
| 试点价 | 500 - 1,000元 |

### Discovery — 发现模式

| 项目 | 内容 |
|------|------|
| 定位 | 初创团队，从零开始 |
| 输入 | 聊天式采访，AI翻译成品牌语言 |
| 输出 | 品牌方向建议 + 基础手册 |
| 包含 | 品牌分析 + 方向建议 |
| **建议价格** | **500 - 1,000元/次** |
| 试点价 | 免费 - 500元 |

## 试点期间调整策略

```
1. 前 3 次试点：灵活定价
   - 可免费或象征性收费
   - 目标是收集反馈，不是收入

2. 收集数据后调整
   - 客户愿意支付多少？
   - 什么模式最受欢迎？
   - 什么行业付费意愿最高？

3. 正式定价
   - 基于试点数据确定最终价格
   - 预计 v1.0 正式发布时确定
```
