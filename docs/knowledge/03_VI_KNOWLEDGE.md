## 企业级 VI 的核心骨架

所有专业 VI 手册都包含以下固定模块，不因品牌类型而改变：

```
Logo 规范
├── 标识诠释（Logo 的含义）
├── 标识网格比例（精确的数学比例）
├── 最小尺寸和保护空间
├── 标识颜色（标准色/单色/反白）
├── 标识背景控制（浅色/深色/图片背景）
├── 标识错误用法（禁止项清单）
└── 标识在版面中的尺寸规范

颜色体系
├── 品牌主色
├── 辅助色
├── 色彩优先级
└── 色彩搭配原则

字体体系
├── 品牌专用字体
├── 通用替代字体
└── 字号层级规范

应用场景
├── 办公系统
├── 资料封面
├── 社媒头像
├── 海报与印刷品
├── 产品包装
├── 空间与门店
└── 数字界面
```

## Logo 规范的关键要素

### 标识网格比例

- Logo 必须有精确的网格比例（以 X 为单位）
- 任何使用中不得改变形状、结构和比例
- 必须使用源文件，不得自行创造或重绘

### 最小尺寸

- 确保 Logo 在所有尺寸下清晰可辨
- 印刷品和数字媒体各有不同的最小尺寸

### 保护空间（Safe Area）

- Logo 周围需要不可侵犯的留白区域
- 通常以 Logo 的高度或宽度作为单位度量

### 背景控制

四种典型背景下的 Logo 使用规则：

| 背景类型 | 规则 |
|---------|------|
| 浅色背景 | 使用标准色 Logo |
| 深色背景 | 使用反白 Logo |
| 图片背景 | 必须确保可识别性，必要时加半透明底 |
| 同色背景 | 避免使用相同颜色 |

## 颜色体系规范

### 三层颜色结构

| 层级 | 用途 | 比例 |
|------|------|------|
| 主色 | 品牌核心色，用于标题、主视觉 | ~50% |
| 辅助色 | 次要元素、信息层级区分 | ~30% |
| 强调色 | CTA、高亮、重要提示 | ~10-20% |

### 颜色必须提供多平台色值

- CMYK（印刷）
- HEX/RGB（数字）
- Pantone（专色）

## 字体体系规范

### 两层字体结构

| 层级 | 用途 | 要求 |
|------|------|------|
| 专用字体 | 品牌 Logo、标题、核心文案 | 需获得授权/购买 |
| 通用字体 | 正文、说明、内部文档 | 必须商用安全 |

### Brand Brain 字体知识库

参见 `src/lib/font-library.ts`，当前收录 10 款商用安全字体：

- 中文：思源黑体 / 思源宋体 / 阿里巴巴普惠体 / HarmonyOS Sans / OPPO Sans
- 英文：Inter / Montserrat / Poppins / IBM Plex / Roboto

## 错误用法清单（必须包含）

一本专业 VI 手册必须明确展示错误用法。常见的错误类型：

1. **变形** — 拉伸、压缩、倾斜
2. **改色** — 使用非标准颜色
3. **加效果** — 投影、渐变、描边
4. **改字体** — 替换 Logo 中的字体
5. **破坏比例** — 任意缩放其中一个元素
6. **移除元素** — 删除任何原始组成部分
7. **添加元素** — 在 Logo 旁加无关图形
8. **错误背景** — 在低对比度背景上使用

## 应用场景的完整性

一个完整的 VI 手册应覆盖以下应用场景（按优先级）：

| 优先级 | 场景 | 说明 |
|--------|------|------|
| P0 | 办公系统 | 名片、信封、PPT |
| P0 | 社媒头像 | 微信、抖音、LinkedIn |
| P1 | 资料封面 | 方案书、报价、白皮书 |
| P1 | 海报 | 活动海报、招聘海报 |
| P2 | 包装 | 产品包装、礼品袋 |
| P2 | 门店 | 门头、空间标识 |
| P3 | 数字界面 | App 图标、网页 Header |
| P3 | 视频 | 视频开场/结尾标识动画 |

