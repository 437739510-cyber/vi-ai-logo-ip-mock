## IP 设定的核心要素

### Identity（身份）

IP 必须有明确的身份定义，回答四个问题：

| 问题 | 示例（小智猫） |
|------|---------------|
| 在品牌中扮演什么角色？ | 首席 AI 增长官 |
| 代表什么能力？ | 数据洞察、策略生成、增长陪跑 |
| 给用户什么感受？ | 聪明、可靠、温暖、行动导向 |
| 和品牌什么关系？ | 品牌人格化入口 |

### Mission（使命）

一句话说清 IP 的职责。"让增长可执行"、"让 AI 有温度"

### Emotion（情感）

IP 的核心情感价值——它不是功能说明书，而是情感连接。

### Memory（记忆点）

- 高频视觉符号（黑屏、像素眼、品牌蓝）
- 让用户一眼认出品牌

## 造型基因拆解（FORM GENE）

一个 IP 通常由 3-5 个核心视觉元素构成。例如小智猫：

| 元素 | 作用 | 可延展性 |
|------|------|---------|
| 猫耳轮廓 | 第一识别点 | 平面/玩偶/数字头像都能识别 |
| 智能面屏 | AI 属性 | 表情变化、交互反馈 |
| 像素眼睛 | 数字生命感 | 加载/分析/提醒/成功等动态语义 |
| 项圈铃铛 | 陪伴属性 | 让 AI 有温度 |
| 增长标识 | 品牌承诺 | 不可移除的关键符号 |

**原则：** 每个元素必须有"为什么存在"的解释，而不仅是为了好看。

---

## 三视图规范

| 视角 | 重点 | 用途 |
|------|------|------|
| 正面 | 主识别特征完整展示 | 品牌传播、物料 |
| 侧面 | 体块厚度、面部弧度 | 动画转身、玩偶生产 |
| 背面 | 简化处理 | 完整建模 |

### 三视图的四个原则

1. **统一体块** — 头部占视觉重心，身体保持圆润稳定
2. **核心符号优先** — 第一识别元素必须保留
3. **量产友好** — 避免复杂尖角和过密细节
4. **延展一致** — 插画/动画/贴纸/物料共享同一结构比例

---

## 色彩系统

IP 颜色与品牌色的关系：

- IP 主色 = 品牌主色（保持统一）
- IP 强调色 = 品牌辅助色（信息反馈）
- IP 留白 = 品牌安全区域

小智猫的色彩比例参考：主色 48% / 深色 27% / 留白 15% / 强调色 10%

---

## 辅助图形系统

IP 的视觉语言应从单一角色延展为完整品牌资产库：

- **数据网格** — 科技背景、规范页
- **像素点阵** — 标题装饰、动态图标
- **增长箭头** — 业务成果、行动按钮
- **AI 光弧** — 封面、活动 KV
- **信息面板** — 数据展示、报告页

---

## 好 IP 的五条验证标准

1. **一眼认出品牌** — 不写名字也知道是谁家的
2. **三句话能说清设定** — 不是长篇大论
3. **缺了任何一个核心元素就不完整** — 每个元素都有功能
4. **能放进不同场景仍保持识别** — 平面/玩偶/动画/物料
5. **和品牌人格一致** — IP 人格是品牌人格的具象化
