# 03 — Layout Engine 规范 (Layout Engine Spec)

> 更新：2026-06-01
> 作用：回答「设计规则是什么」——未来核心资产

---

## 核心分工

```
AI（DeepSeek / Manus / GPT）→ 内容
Layout Engine → 设计
```

## Logo 规则

- **Logo 永不重绘** — 只能缩放、移动、排版围绕。不能改色、加效果、替换字体
- 封面 Logo：页面宽度的 30%-50%，居中偏下
- 内页 Logo：页面宽度的 6%-15%，左上角
- Logo 周围保留至少等于 Logo 高度 30% 的安全空间
- 背景适配：浅色背景用彩色 Logo，深色背景用反白 Logo，图片背景加半透明遮罩

## IP 规则

- **已有 IP 永不重绘** — 不能改脸型、颜色、材质、比例
- IP 尺寸由版面决定，不变形即可（参考案例：蜜雪冰城雪王可占页面 70%）
- IP 与 Logo 共存时根据版面灵活调整，无固定比例限制

## 图片规则

| 用途 | 来源 | 处理方式 |
|------|------|---------|
| Logo/IP | 品牌方上传 | 原图引用，不修改 |
| 背景图 | 通义万相生成 | 无文字、无 Logo |
| 装饰图 | 预留接口（Flux/DALL·E） | 统一 Provider |

## 留白规则

- 页面边距：8%-15%（按页面类型）
- 标题与正文间距：页面高度的 3%-5%
- 高端品牌留白多，大众品牌留白适度

## 字体规则

| 层级 | 用途 | 字号 (pt) |
|------|------|----------|
| H0 | 封面标题 | 48-72 |
| H1 | 页面标题 | 24-36 |
| H2 | 章节标题 | 16-22 |
| Body | 正文 | 9-12 |
| Caption | 标注 | 7-9 |

- 中文字体方案：Noto Sans SC / 思源黑体（待内置）
- 正文最小字号 9pt，行高 1.5 倍

## 色彩规则

- 品牌主色用于标题/关键信息（5%-10%）
- 品牌强调色用于点缀（< 5%）
- 正文使用深灰（#333-666）确保可读性
- 对比度最低要求 WCAG AA（正文 4.5:1）
