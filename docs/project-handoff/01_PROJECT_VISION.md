## 一句话定位

**Brand Brain = AI 品牌顾问系统**

不是 AI Logo 生成器。不是 AI 图片生成器。不是 VI 生成器。

而是帮助企业完成：

```
品牌诊断
品牌策略
品牌资产规划
VI 手册生成
```

## 核心理念

```
AI 负责理解品牌
Layout Engine 负责设计
```

- AI（DeepSeek / Manus / GPT 等）是可替换的
- Layout Engine 是不可替换的——它是品牌的设计规则系统
- 所有模型输出的内容必须经过 Layout Engine 的设计规则约束

## 最大差异化

```
当前市场竞争：AI Logo 生成器（量大、同质化）
Brand Brain 定位：AI 品牌顾问（深度、定制化、专业）
```

客户买的是品牌方案，不是图片。老师案例、Manus 案例、椰岛工坊案例都证明了这一点。
