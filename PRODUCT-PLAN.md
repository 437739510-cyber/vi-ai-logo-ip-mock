# 线上代客生成企业VI手册 — 产品规划书

> **商业模式**：B2B 代运营服务。客户仅通过极简界面提交需求与素材，VI 手册由我方团队（含 AI 辅助）生成并交付。

---

## 一、产品定位

AI 辅助的**企业 VI 代运营平台**。客户提交品牌 Logo、IP 公仔及需求信息，我方通过 AI 分析 + 人工精修，生成完整的品牌视觉识别系统（VI）手册并交付。核心价值在于**降低客户沟通成本、缩短 VI 设计周期**。

---

## 三、客户端页面（客户可见）

### 3.1 首页 (Homepage)
- 服务价值展示：案例轮播、服务流程示意、信任背书
- CTA：立即咨询 / 开始提交需求

### 3.2 咨询留资 (Consultation)
- 表单字段：姓名、公司名称、行业、联系方式（电话/微信）、预算范围、需求描述
- 提交后触发内部通知（企微/邮件）

### 3.3 LOGO 上传
- 支持格式：PNG / JPG / SVG / AI / EPS
- 自动展示预览缩略图
- 可上传多个版本（彩色版、反白版）

### 3.4 IP 公仔 / 吉祥物上传
- 支持格式：PNG / JPG / GIF（多角度更佳）
- 字段填写：IP 名称、性格描述（可选）
- 可上传多个角度/动作的图片

### 3.5 参考 VI 手册上传（新增）
- **功能**：客户可上传已有的 VI 手册 / 品牌指南 PDF，供 AI 参考风格
- **说明文案**："如果您已有品牌倾向的视觉参考手册，请上传，AI 会优先学习您偏好的风格"
- 支持格式：PDF（建议不超过 50MB）
- AI 侧处理：提取 PDF 中的色板、字体、版式比例、图形语言
- 可选是否参考，非必填

### 3.6 需求确认页
- 汇总展示所有上传素材 + 填写信息
- 确认提交后生成项目号，给出"我们将在 X 个工作日内联系您"的反馈

### 3.7 （可选）项目进度查询
- 凭项目号 + 手机号查询当前状态：已提交 / AI 分析中 / 设计中 / 待交付 / 已交付

## 四、管理端页面（我方使用）

### 4.1 工作台 (Dashboard)
- 待处理项目数、进行中项目数、本月交付数
- 最新咨询留资列表（带跟进状态）

### 4.2 项目管理
- 项目列表：按状态筛选（待处理 / AI 分析中 / 设计制作 / 待客户确认 / 已交付）
- 每个项目可查看客户提交的全部素材和需求

### 4.3 素材管理
- 查看 / 下载客户上传的 Logo、IP 公仔、参考 VI 手册
- 对素材打标签（颜色提取结果、风格倾向等）

### 4.4 AI 方案生成
- 选择项目后，点击「AI 生成方案」：
  - 可勾选 **「参考客户上传的 VI 手册」**（若客户提供了）
  - 可额外上传**我们自己找的参考图/手册**（内部参考素材库）
  - AI 生成 3–5 套方案供选择
- 支持多轮生成（换 seed / 换风格方向重新生成）

### 4.5 VI 在线编辑器
- 对 AI 生成的方案进行微调：品牌色板、字体、间距、版式
- 左侧工具栏 + 右侧 A4 比例实时预览

### 4.6 AI 对话助手
- 嵌入编辑器的 AI 对话窗口
- 输入如"把这个蓝色调暖一些"、"生成一页名片应用规范"等指令

### 4.7 手册预览 & 批注
- 完整 VI 手册翻页预览
- 支持添加批注（内部审核用）

### 4.8 收藏方案
- 在 AI 生成的多个方案中收藏喜欢的，供后续复用或作为模板

### 4.9 导出交付
- 格式：PDF（300DPI 打印级）、PPT（可编辑源文件）、Web 展示链接
- 交付后可一键通知客户（短信 / 邮件）

### 4.10 客户管理
- 客户列表：关联的项目、历史咨询记录、跟进状态

## 五、新增核心功能：参考 VI 手册方案生成

这是你特别提到的，详细说明：

### 5.1 触发流程
```
客户上传参考手册（PDF）
  ↓
AI 解析 PDF 内容：
  · 品牌色 → 提取 HEX/RGB/CMYK
  · 字体 → 识别字体族、字重
  · 版式 → 分析网格系统、留白比例
  · 图形语言 → 分析辅助图形风格
  ↓
管理端「生成方案」时，勾选 ☑ 参考此手册
  ↓
AI 在生成时参考上述特征，产出风格一致的方案
```

### 5.2 参考模式选项
| 模式 | 说明 |
|---|---|
| **强参考** | 严格遵循参考手册的色彩和字体规范，只做微调 |
| **弱参考** | 提取参考手册的风格倾向，但允许 AI 做更大胆的变化 |
| **不参考** | 完全基于客户 Logo/IP 和行业属性从零生成 |

### 5.3 内部素材库
- 我们也积累一个**内部参考 VI 库**，按行业/风格分类
- 生成时可选"参考客户手册" + "参考内部库"双源输入

## 七、推荐技术栈

### 7.1 前端（双端共用一套代码，通过路由权限区分）

| 类别 | 推荐 | 理由 |
|---|---|---|
| 框架 | **Next.js 15** (React) | SSR 提升首页 SEO，App Router 清晰分端 |
| 样式 | Tailwind CSS 4 | 快速原型 |
| UI 组件 | shadcn/ui | 专业感、可访问性好 |
| 状态管理 | Zustand | 轻量 |
| PDF 解析 | pdf.js (客户端预览) + PyMuPDF (服务端提取) | 手册分析 |
| 图片上传 | uppy / react-dropzone | 大文件分片上传 |

### 7.2 AI / 后端

| 类别 | 推荐 | 理由 |
|---|---|---|
| AI 视觉分析 | GPT-4o (Vision) | Logo 分析、风格提取、色板提取 |
| PDF 内容提取 | PyMuPDF / LlamaParse | VI 手册结构化解析 |
| AI 生成方案 | Claude Sonnet 4 + DALL·E 3 | 风格推理 + 辅助图形生成 |
| 后端 | Python FastAPI | 适合 AI 密集任务 |
| 数据库 | PostgreSQL | 结构化存储 |
| 文件存储 | S3 / R2 | Logo、IP、手册素材 |

### 7.3 DevOps

| 类别 | 推荐 |
|---|---|
| 部署 | Vercel (前端) + Railway/Fly.io (后端) |
| 文件上传 | 直接上传到 R2（预签名 URL） |
| 任务队列 | Celery / Redis Queue | AI 生成异步处理 |

## 九、未来可扩展点

### Phase 2（1–3 个月）

- **内部 VI 参考素材库** — 积累按行业/风格分类的高质量 VI 手册库，AI 生成时多源参考
- **自动 Logo 变体生成** — 从客户 Logo 自动生成反白、灰度、缩小版等规范变体
- **客户项目看板开放** — 可选开放部分进度看板给客户（非必须）
- **多语言交付** — 手册同时输出中英文版
- **企微/飞书/钉钉集成** — 留资自动推送到内部协作工具

### Phase 3（3–6 个月）

- **AI 商标近似检测** — 生成前扫描与注册商标的近似度，降低侵权风险
- **批量交付** — 一次生成多个客户的 VI 手册（活动批量单场景）
- **品牌资产托管** — 交付后持续为客户管理品牌素材更新
- **印刷下单直连** — 手册/名片可直接下单印刷
- **短视频版 VI 展示** — 将手册转为 30s 展示视频供客户社交媒体使用

### Phase 4（6–12 个月）

- **AI 品牌全案升级** — 从 VI 延伸到品牌命名、Slogan、品牌故事
- **行业垂直模型** — 针对餐饮、零售、科技等领域微调 VI 生成模型
- **API 批发能力** — 开放给设计公司 / 广告代理商用白标模式
- **C 端轻量版** — 面向个人创作者的低成本自助入口（不冲突 B2B 主线）
- **3D 品牌展厅** — 客户品牌 VI 的 Web 3D 场景展示

## 十一、设计基调

- **客户端**：温暖、可信赖 — 让客户感到安心地把品牌交给我们。柔和色块 + 清晰的信息层级 + 友好的文案语调
- **管理端**：高效、专业 — 深色侧边栏 + 卡片式列表，信息密度高，操作路径短
- **手册风格本身**：根据客户品牌行业动态适配，不预设固定风格
